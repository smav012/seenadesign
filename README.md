# Seena's Design Portfolio

Modern, responsive portfolio website for Seena Mavaddat.

## What We've Added

### 1. Responsive CSS (`css/responsive.css`)
- Mobile-first responsive improvements
- Better typography scaling for mobile/tablet
- Improved image handling on mobile
- Better gallery layouts on mobile
- Modern hover effects that work with your yellow theme
- Smooth scrolling for the whole site
- Improved accessibility (focus states, selection colors)
- Print styles
- Dark mode support
- Custom scrollbar matching your theme
- Better form elements if needed

### 2. Modern JavaScript Enhancements (`js/enhancements.js`)
- Smooth scrolling for anchor links
- Lazy loading for images
- Improved focus visibility for keyboard users
- Subtle hover effects on project items
- Print-friendly improvements
- Touch device detection
- Responsive image handling
- Form accessibility improvements
- Performance optimizations

## How It Works

These enhancements work alongside your existing Cargo.site template without changing your layout or design language. They add:

1. **Better mobile experience** - Your site will look great on phones and tablets
2. **Modern interactions** - Smooth scrolling, better hover effects
3. **Accessibility improvements** - Better for keyboard users and screen readers
4. **Performance optimizations** - Lazy loading, debounced scroll events

## Synergy with Your Art Portfolio (seena.site)

For synergy between your design portfolio (seena.design) and art portfolio (seena.site), consider:

1. **Consistent branding** - Use similar yellow highlights and typography
2. **Cross-linking** - Clear navigation between both sites
3. **Shared components** - Reuse responsive CSS and JavaScript
4. **Unified contact info** - Same email/Instagram links

## Next Steps for Moving Off Cargo.site

When you're ready to move off Cargo.site, you can:

1. **Export your content** - Save all your project images and text
2. **Create a clean HTML structure** - Build a simple, semantic HTML file
3. **Transfer the responsive CSS** - Move our responsive.css to your new site
4. **Keep the JavaScript enhancements** - Transfer enhancements.js
5. **Add modern meta tags** - Viewport, Open Graph, etc.
